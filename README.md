Employee Data Management System

This is a simple Employee Management System built using Python.
The program allows the user to manage employee records using a menu-based command-line interface.

The project is designed to practice core Python concepts such as OOP, dictionaries, CSV file handling, and basic validation.

Project Features

Add a new employee (ID, Name, Position, Salary, Email)

View all employees

Update employee details using employee ID

Delete an employee by ID

Search for an employee by ID

Save all data in a CSV file

Load data automatically when the program starts

Technologies Used

Python 3

csv module

Dictionary for in-memory storage

Command Line Interface (CLI)

How the Program Works

When the program starts, employee data is loaded from employees.csv.

The user chooses an option from the menu.

The selected operation is performed (add, update, delete, search, or view).

Any changes are saved back to the CSV file.

The program continues running until the user chooses Exit.
